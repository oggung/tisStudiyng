DOM 노드를 측정하려면 어떻게 해야 합니까? ==> DOM 노드의 위치나 크기를 측정하는 기본적인 방법의 하나는 콜백 ref를 사용하는 것입니다. https://ko.reactjs.org/docs/hooks-faq.html#how-can-i-measure-a-dom-node MeasureSize 컴포넌트 만들기 function MeasureSize() { const [height, setHeight] = useState(0); const measuredRef = useCallback(node => { if (node !== null) { setHeight(node.getBoundingClientRect().height); } }, []); return ( <Fragment> <h1 ref={measuredRef}>Hello, world</h1> <h2>The above header is {Math.round(height)}px tall</h2> </Fragment> ); }
